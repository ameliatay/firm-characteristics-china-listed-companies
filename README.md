## Set up (One time)
- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `pip3 install streamlit`
- `pip3 install plotly`
- `pip3 install pandas`
- `pip3 install streamlit_option_menu`

## Running the app locally
- `streamlit run app.py`
